Winter Is Comming
=================

Little 3D Viewer with Chritmas theme using Python & OpenGL.
3D models and textures are the property of TeamTO Studio.
This application uses legacy OpenGL. It advice to don't take example on this code.

Dependencies:
- OpenGL version 2.X
- Python 2.7
- Python 2.7 packages:
	- PyOpenGL
	- numpy
	- pygame
